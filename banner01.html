These test cases are designed to determine how the header element and elements with a role of banner are announced by various screen readers, in specific situations.

By default, the element being tested does not receive focus, so the only way to test how it is announced is via:

  • VoiceOver: Announce next item: VO + Right Arrow
  • NVDA and JAWS: Announce next line: DOWN ARROW

Example

Test content

Code

<header>
  Test content
</header>
