role="banner" inside <article>

Return to main list

This test case is designed to determine how the role="banner" will be announced by various screen readers when inside the <article> element.

In this case, the element should not be announced as a banner landmark.

By default, the element being tested does not receive focus, so the only way to test how it is announced is via:

  • VoiceOver: Announce next item: VO + Right Arrow
  • NVDA and JAWS: Announce next line: DOWN ARROW

Example

Article content
Test content

Code

<article>
  Article content
  <div role="banner">
    Test content
  </div>
</article>
