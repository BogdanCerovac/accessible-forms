role="banner" descendant of <article>

Return to main list

This test case is designed to determine how the role="banner" will be announced by various screen readers when it is a descendant of the <article> element.

In this case, the element should not be announced as a banner landmark, as it is not scoped to the <body> element.

By default, the element being tested does not receive focus, so the only way to test how it is announced is via:

  • VoiceOver: Announce next item: VO + Right Arrow
  • NVDA and JAWS: Announce next line: DOWN ARROW
