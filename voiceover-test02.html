VoiceOver test 02 - with ARIA

Return to main list

This test case is designed to determine how landmark elements are displayed in VoiceOver's Rotor Landmark list.

Example

1. Header content
2. Main content
3. Aside content
4. Nav content
5. Article content
6. Footer content

Results

HTML element Announced Role displayed in Rotor PASS/FAIL
<header role="banner"> "Banner", "End of banner" "banner" PASS
<"main"> "Main", "End of main" "main" PASS
<aside> "Complementary", "End of complementary" "complementary" PASS
<nav> "Navigation", "End of navigation" "navigation" PASS
<article> "Article", "End of article" "article" PASS
<footer role="contentinfo"> "Footer", "End of footer" "footer" PASS

Code

<header role="banner">
  1. Header content
</header>
<main>
  2. Main content
</main>
<aside>
  3. Aside content
</aside>
<nav>
  4. Nav content
</nav>
<article>
  5. Article content
</article>
<footer role="contentinfo">
  6. Footer content
</footer>
