Button accessible name tests

The purpose these tests is to show the cascade order of how accessible names are applied in the accessibility tree for the <button> element. These examples should not be considered proper practice.

  • → <button> where "aria-labelledby" should be accessible name
  • → <button> where "aria-label" should be accessible name
  • → <button> where content should be accessible name
  • → <button> where "title" should be accessible name
  • → <button> with no accessible name

ARIA banner tests - Added 10/07/2021

When the <header> or an element with role=banner" is scoped to the <body> element, it represents introductory content for the page as a whole. In these cases, the element will have an implicit ARIA role of banner.

The element will not have an implicit role of banner if it is a descendant of:

  • <article> or role="article"
  • <aside> or role="complementary
  • <main> or role="main"
  • <nav> or role="navigation"
  • <section> or role="region"

These test cases are designed to test the situations listed above.

No descendant

  • → <header>
  • → role="banner"

Descendant of <article>

  • → <header> descendant of <article>
  • → role="banner" descendant of <article>

Descendant of role="article"

  • → <header> descendant of role="article"
  • → role="banner" descendant of role="article"

Descendant of <aside>

  • → <header> descendant of <aside>
  • → role="banner" descendant of <aside>

Descendant of role="complementary"

  • → <header> descendant of role="complementary"
  • → role="banner" descendant of role="complementary"

Descendant of <main>

  • → <header> descendant of <main>
  • → role="banner" descendant of <main>

Descendant of role="main"

  • → <header> descendant of role="main"
  • → role="banner" descendant of role="main"

Descendant of <nav>

  • → <header> descendant of <nav>
  • → role="banner" descendant of <nav>

Descendant of role="navigation"

  • → <header> descendant of role="navigation"
  • → role="banner" descendant of role="navigation"

Descendant of <section>

  • → <header> descendant of <section>
  • → role="banner" descendant of <section>

Descendant of role="region"

  • → <header> descendant of role="region"
  • → role="banner" descendant of role="region"
