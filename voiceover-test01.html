VoiceOver test 01 - without ARIA

Return to main list

This test case is designed to determine how landmark elements are displayed in VoiceOver's Rotor Landmark list.

Example

1. Header content
2. Main content
3. Aside content
4. Nav content
5. Article content
6. Footer content

Results

HTML element Role displayed in Rotor PASS/FAIL
<header> Not announced FAIL
<"main"> "main" PASS
<aside> "complementary" PASS
<nav> "navigation" PASS
<article> "article" PASS
<footer> Not announced FAIL

</header>

</main>
