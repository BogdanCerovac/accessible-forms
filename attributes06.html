input with "min" and "aria-valuemin" attribute

This is a test case designed to determine how the min and aria-valuemin attributes are announced. In real situations, authors should always provide clear instructions on how to successfully complete form questions.

Example

Code

<div>
  <label for="number">Choose a number</label>
  <input 
    type="number"
    id="number"
    aria-describedby="help-text"
    min="5"
    aria-valuemin="5"
  >
</div>
