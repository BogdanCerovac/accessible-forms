role="banner" inside role="article"

Return to main list

This test case is designed to determine how the role="banner" will be announced by various screen readers when inside an element with role="article". In this case, role="banner" should not be announced as a banner landmark, as it is not scoped to the <body> element.

By default, the element being tested does not receive focus, so the only way to test how it is announced is via:

  • VoiceOver: Announce next item: VO + Right Arrow
  • NVDA and JAWS: Announce next line: DOWN ARROW

Example

Article content
Test content

Code

<div role="article">
  Article content
  <div role="banner">
    Test content
  </div>
</div>
